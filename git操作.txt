如何将本地的仓库上传到github仓库中
首先先新建一个空的文件夹
在空的文件夹中打开git bash 右键git bash 
然后再敲入git init  然后git add 文件 这步操作是将这个文件放入到仓库的暂存区 如果添加的文件和修改的文件很多可以直接git add . 将所用的文件放入到暂存区中
git commit -m '所有的计算机基础的ppt'  
git log查看提交的文件的日志  有一个commit id 03eee2bb790c05cfe492bfd3e199cbb32e878e87 
git log --stat 查看每次提交了哪些修改文件  
配置验证登录ssh-keygen -t rsa -C 1931665318@qq.com 一直按enter 然后到/c/Users/asus/.ssh/id_rsa.pub找到公钥回到github中点击头像然后setting 找到ssh new sshkey 随便取一个名字然后把id_rsa.pub中的内容复制上去
然后找到你要上传的仓库git@github.com:Bais-art/teaching.git    然后 git remote add teaching git@github.com:Bais-art/teaching.git 一旦配置了ssh 以后别的上传别的仓库也不需要配置ssh了。
最后git push teaching master 作为主干上传上去
